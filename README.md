# Build_tools_test

## 概要
- 各種ビルドツールの使用方法のテストコードを実装
- 基本的にUbuntu22.04でのみ動作確認

## ディレクトリ構成
- [./autoconf](./autoconf/)では`autoconf`の使用方法をテスト
- [./bazel](./bazel/)ではGoogleのBazelの使用方法をテスト
- [./cmake](./cmake/)ではCMakeの使用方法をテスト
- [./makefile](./makefile/)ではMakefileの使用方法をテスト
